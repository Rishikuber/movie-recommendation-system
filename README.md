# 🎬 Movie Recommendation System

A Movie Recommendation System built using Python, Machine Learning, and FastAPI.

The system recommends similar movies based on user rating patterns using collaborative filtering and cosine similarity.

---

## 🚀 Features

- Movie similarity calculation
- Recommendation API using FastAPI
- Interactive API documentation
- Machine learning based recommender

---

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- FastAPI
- Uvicorn
